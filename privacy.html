📌 핵심 요약

  • 카메라 영상은 기기 내부에서만 처리되며 외부 서버로 전송·저장되지 않습니다.
  • 회원가입이 없으며 이름·이메일·전화번호 등 개인 식별 정보를 수집하지 않습니다.
  • 운동 기록은 사용자 기기에만 저장되며, iCloud 백업 외에는 외부로 동기화되지 않습니다.

  • 광고 식별자(IDFA)를 사용하지 않으며, 제3자 추적 SDK가 포함되어 있지 않습니다.
본 문서 안내: 푸시업 옵저버(이하 "본 앱")는 사용자의 개인정보를 소중히 다루며, 「개인정보 보호법」을 비롯한 관련 법령을 준수합니다. 본 방침은 본 앱이 어떤 정보를 어떻게 처리하는지 투명하게 설명합니다.

01총칙

TODO: 사업자명(이하 "운영자")가 제공하는 푸시업 옵저버 앱(이하 "본 앱")은 이용자의 개인정보를 보호하기 위해 「개인정보 보호법」 등 관련 법령에 따라 본 개인정보처리방침을 수립·공개합니다.

본 앱은 카메라를 활용한 자세 분석 기능을 제공하며, 모든 분석은 기기 내부의 머신러닝 모델로 수행됩니다. 영상 데이터가 운영자 또는 제3자의 서버로 전송되지 않습니다.

02수집하는 정보 및 권한

2.1 본 앱이 사용하는 기기 권한

권한 사용 목적 필수 여부
카메라 운동 자세 실시간 분석 (영상은 기기 내부에서만 처리, 외부 전송 없음) 필수
로컬 네트워크 멀티 디바이스 연동 시 같은 네트워크 내 iPad·Mac과의 P2P 통신 선택
사진 (저장) "영상 녹화" 옵션 사용 시 운동 영상을 사용자 사진 보관함에 저장 선택
마이크 "영상 녹화" 옵션 사용 시 음성 함께 녹음 (사용자가 비활성화 가능) 선택

2.2 수집하지 않는 정보

본 앱은 다음 정보를 수집하지 않습니다.

  • 이름, 이메일, 전화번호, 주소 등 개인 식별 정보
  • 위치 정보 (GPS·Wi-Fi 위치)
  • 광고 식별자(IDFA), 디바이스 추적 식별자
  • 연락처, 캘린더, 사진 보관함의 기존 콘텐츠
  • 건강 데이터(HealthKit) 현재 미연동, 추후 추가 시 본 항목 갱신 예정

2.3 자동으로 생성되는 정보

다음 정보는 사용자의 기기에만 저장되며 운영자에게 전송되지 않습니다.

  • 운동 기록: 운동 종목, 횟수, 시간, 달성률, 측정 일시
  • 녹화 영상: 사용자가 직접 활성화한 경우에만 생성, 기기 내 사진 보관함에 저장
중요: 위 정보는 모두 사용자 기기 내부에 보관됩니다. iCloud 백업이 활성화된 경우 사용자의 iCloud 계정에 암호화되어 백업될 수 있으며, 이는 Apple의 정책에 따릅니다.

03정보의 이용 목적

본 앱은 다음 목적으로만 정보를 이용합니다.

  1. 운동 자세 분석 및 카운트: 카메라 영상을 통해 사용자의 관절 각도를 측정하고 횟수를 자동 계산
  2. 운동 기록 제공: 사용자가 자신의 운동 이력을 확인할 수 있도록 기기에 저장
  3. 멀티 디바이스 연동: 사용자의 다른 기기(iPad·Mac)에 실시간 영상과 운동 데이터 전송
  4. 운동 영상 저장: 사용자가 옵션을 활성화한 경우 자세 점검을 위해 사진 보관함에 저장

07정보주체의 권리

이용자는 「개인정보 보호법」 제35조에 따라 다음의 권리를 행사할 수 있습니다.

  • 열람·확인: 본 앱의 "기록" 탭에서 본인의 운동 기록을 직접 확인 가능
  • 수정·삭제: 운동 기록은 "기록" 탭의 Edit 버튼으로 직접 삭제 가능
  • 처리 정지: 앱 삭제로 모든 데이터 처리를 즉시 중단할 수 있음

본 앱은 운영자의 서버에 사용자 정보를 보관하지 않으므로, 별도의 열람·정정·삭제 청구 절차가 필요하지 않습니다. 모든 데이터는 사용자가 직접 통제합니다.

08정보의 파기

본 앱은 사용자의 정보를 다음과 같이 파기합니다.

  • 운동 기록 삭제: 사용자가 "기록" 탭에서 개별 또는 전체 삭제 시 즉시 파기
  • 앱 삭제: iOS의 앱 삭제 기능으로 본 앱과 관련된 모든 로컬 데이터가 즉시 파기
  • 녹화 영상: 사진 앱에서 사용자가 직접 삭제 (애플 정책에 따라 휴지통 보관 후 영구 삭제)

09안전성 확보 조치

운영자는 이용자의 정보를 안전하게 보호하기 위해 다음과 같은 조치를 취하고 있습니다.

  1. 온디바이스 처리: 자세 분석 ML 모델이 사용자 기기에서 직접 동작하므로 영상이 외부로 노출되지 않습니다.
  2. iOS 시스템 보안: 사용자 데이터는 Apple이 제공하는 iOS의 샌드박스 환경에서 암호화되어 저장됩니다.
  3. 암호화 통신: 멀티 디바이스 연동 시 같은 사용자 소유 기기 간에만 암호화된 채널로 데이터가 전송됩니다.
  4. 최소 권한 원칙: 앱 동작에 필요한 최소한의 권한만 요청합니다.
