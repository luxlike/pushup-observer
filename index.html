02 / AI 자세 분석

관절을 보는 AI의 시선

On-device 머신러닝 모델이 14개 핵심 관절을 추적합니다. 서버로 영상이 전송되지 않으므로 빠르고, 안전합니다.

  • 01 측면 90° 자세 가이드 운동 시작 전 가이드 오버레이로 카메라 위치를 정확하게 잡을 수 있습니다.
  • 02 실시간 관절 각도 표시 팔꿈치·무릎·고관절 등 운동에 필요한 각도가 화면에 즉시 표시됩니다.
  • 03 잘못된 자세는 카운트 제외 가동 범위가 부족한 동작은 카운트에서 자동으로 제외되어 정직한 기록이 남습니다.
  • 04 영상 녹화 옵션 원하는 경우 운동 영상을 함께 녹화해서 자세를 직접 점검할 수 있습니다.
03 / 운동 기록

매일의 기록이 습관을 만든다

운동을 끝낼 때마다 자동으로 기록이 저장됩니다. 숫자가 쌓일수록 다음 운동에 대한 동기가 생깁니다.

  • 01 종합 통계 총 운동 횟수, 총 반복 횟수, 목표 달성률을 한 눈에 확인합니다.
  • 02 최근 7일 그래프 일자별 운동량을 막대 차트로 시각화하고 목표선을 함께 표시합니다.
  • 03 운동 히스토리 날짜·운동 종목·횟수가 시간 순으로 정렬되어 언제든 다시 볼 수 있습니다.

04 / 멀티 디바이스 연동

iPhone에서 찍고, iPad·Mac에서 본다

큰 화면으로 자기 자세를 확인하고 싶을 때, iPhone을 송신 모드로 설정하고 iPad나 Mac을 수신 모드로 두면 실시간 영상과 운동 데이터가 동시에 전송됩니다.

  • → iPhone 송신 모드 운동 탭의 송신 버튼을 누르면 다른 기기로 카메라 영상을 전송합니다.
  • ← iPad / Mac 수신 모드 큰 화면에서 자신의 자세를 직접 보면서 운동할 수 있습니다.
  • + 15fps 영상 + 운동 데이터 실시간 영상과 함께 횟수·각도·피드백 정보가 함께 전송됩니다.
05 / 사용 방법

처음 사용하는 5단계

앱을 처음 열었을 때 이렇게 따라 하시면 첫 운동을 빠르게 시작할 수 있습니다.

📱

기기를 측면에 거치

매트의 옆쪽에 iPhone을 세로 또는 가로로 거치합니다. 카메라가 몸의 측면을 정면으로 바라보도록 90° 각도를 맞춰 주세요.

바닥에서 30~50cm 정도 높이가 가장 잘 인식됩니다.
🎯

운동 종목과 목표 선택

상단 탭에서 푸시업·스쿼트·플랭크 중 하나를 선택하고, 좌측 상단 "목표" 버튼으로 횟수(또는 시간)를 설정합니다.

목표는 10회/15회/20회 또는 직접 입력으로 설정 가능합니다.
▶

시작 버튼을 눌러 운동

초록색 "시작" 버튼을 누르면 카운트다운 후 측정이 시작됩니다. 화면에는 관절 각도와 현재 횟수가 실시간 표시됩니다.

측정을 멈추려면 빨간색 "중지" 버튼을 누르면 됩니다.
📊

기록 탭에서 결과 확인

하단 "기록" 탭으로 이동하면 종합 통계, 최근 7일 그래프, 전체 운동 히스토리를 볼 수 있습니다.

달성률은 (실제 횟수 ÷ 목표 횟수) × 100 으로 계산됩니다.
📡

큰 화면이 필요할 때

하단 "연동" 탭에서 iPhone 송신 모드를 켜고, iPad나 Mac에서 수신 모드를 켜면 두 기기가 자동으로 연결됩니다.

두 기기가 같은 Wi-Fi 또는 가까운 거리에 있어야 연결됩니다.
🎬

자세 점검용 영상 녹화

하단의 "영상 녹화"를 활성화하면 운동 중 영상이 함께 저장됩니다. 사진 앱에서 다시 보며 자세를 점검할 수 있습니다.

녹화된 영상은 모두 기기 내부에만 저장됩니다.
06 / 자주 묻는 질문

궁금한 것이 있으세요?

카메라 영상이 외부 서버로 전송되나요? +
아닙니다. 자세 분석은 모두 iPhone 내부에서 On-device로 처리됩니다. 영상이 외부 서버로 전송되거나 저장되지 않습니다. 멀티 디바이스 연동 시에도 같은 Wi-Fi 네트워크 내에서 직접 통신합니다.
왜 카운트가 올라가지 않을 때가 있나요? +
몇 가지 원인이 있을 수 있습니다.
  • 측면이 아닌 각도에서 촬영 — 정면이나 위에서 찍으면 관절 각도를 정확히 측정할 수 없습니다.
  • 가동 범위 부족 — 푸시업의 경우 팔꿈치가 충분히 굽혀져야 합니다.
  • 몸의 직선이 무너진 자세 — 엉덩이가 처지거나 솟으면 인식되지 않습니다.
멀티 디바이스 연동에 필요한 조건은? +
송신 기기와 수신 기기가 같은 Wi-Fi 네트워크에 연결되어 있어야 합니다. iPad는 iPadOS 16 이상, Mac은 macOS 13 (Ventura) 이상에서 동작합니다. 두 기기에서 모두 푸시업 옵저버 앱이 설치되어 있어야 합니다.
운동 데이터가 Apple 건강 앱과 연동되나요? +
현재 버전에서는 자체 기록 시스템만 제공합니다. HealthKit 연동은 추후 업데이트에서 추가될 예정입니다. 데이터 보존을 위해 iCloud 백업을 활성화하시는 것을 권장합니다.
카메라가 어두운 환경에서도 인식하나요? +
최소한의 실내 조명은 필요합니다. 너무 어두우면 관절 추적의 정확도가 크게 떨어집니다. 가급적 밝은 환경에서, 운동복은 배경과 대비되는 색상을 권장합니다.
유료 결제가 있나요? +
기본 기능은 무료로 제공됩니다. 추후 고급 통계, 추가 운동 종목 등의 일부 기능에 인앱 결제가 도입될 수 있으나, 핵심인 자세 분석과 카운트 기능은 계속 무료로 유지됩니다.
